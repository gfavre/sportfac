Fonctionnement

Les activités disponibles pour chaque enfant se trouvent à gauche. Une fois l'activité sélectionnée, cliquez sur l'événement dans le calendrier pour inscrire votre enfant. Faites ainsi pour chaque enfant à inscrire.
Lorsque vous avez terminé, appuyez sur le bouton "Terminer l'inscription" au bas de cette page.
